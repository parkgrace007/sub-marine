# 세션 작업 현황

> 이 파일은 여러 Claude 세션 간 작업 충돌을 방지하기 위한 공유 파일입니다.
> 작업 시작 전 반드시 확인하고, 본인 세션의 상태를 업데이트하세요.

---

## 세션 현황

| 세션 | 담당 영역 | 현재 작업 | 상태 |
|------|----------|----------|------|
| Backend | `backend/**` | 대기 중 | idle |
| Frontend | `frontend/**` | 대기 중 | idle |
| API | 조사/문서 | 대기 중 | idle |

---

## 작업 규칙

1. **작업 시작 전**: 이 파일 읽고 충돌 여부 확인
2. **작업 시작 시**: 본인 세션의 "현재 작업"과 "상태" 업데이트
3. **작업 완료 시**: 상태를 `idle`로 변경
4. **긴급 수정 필요 시**: 다른 세션 영역 수정 전 여기에 기록

---

## 상태 값

- `idle` - 대기 중
- `working` - 작업 중
- `blocked` - 다른 세션 대기 중

---

|------|------|------|

## 주의사항

- `backend/` ↔ `frontend/` 간 API 인터페이스 변경 시 양쪽 세션에 알릴 것
- Git commit/push는 한 세션에서만 수행 (충돌 방지)
- 공유 파일 (CLAUDE.md, Docs/*) 수정 시 여기에 기록
